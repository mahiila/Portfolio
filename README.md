#  Personal Portfolio Website

##  Project Overview
This project is a personal portfolio website developed to present academic background, technical skills, projects, achievements, and contact information in a clear and professional format.

##  Objective
The objective of this project is to apply core concepts of **HTML and CSS** to design and implement a complete personal portfolio suitable for academic evaluation and professional presentation.

##  Design (Figma)
The user interface of this portfolio was designed using **Figma** prior to development, following standard UI/UX design practices.

- Responsive single-page layout
- Fixed navigation bar
- Smooth scrolling between sections
- Project showcase with images
- Downloadable CV
- Contact form interface
- Social media profile links

##  Content Sections
- Home
- About
- Education
- Projects
- Achievements
- Hobbies
- Publications
- Contact

##  Project Description
The portfolio highlights several academic and personal projects, including:
- **Movie Ticket Booking System** developed using Qt and C++
- **Hotel Management System** developed using C
- **Pet Management System** developed using C++

These projects demonstrate foundational programming knowledge and problem-solving skills.

## How to Run Locally
1. Clone the repository:
   git clone https://github.com/mahiila/Portfolio.git
2. Navigate to the project folder:
   cd Portfolio
3. Open the index.html file in a web browser.
